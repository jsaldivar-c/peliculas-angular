### **Caracterisiticas**

<br>

En este pequeño proyecto se trabaja con la API de The Movie Database para hacer una aplicación de búsqueda de películas trabajando con:

> -   Http Get
> -   Operadores de RXJS (Tap y Map)
> -   Combinación de observables
> -   Módulos
> -   Servicios
